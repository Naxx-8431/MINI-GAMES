# Mini Games Website

This project contains two small browser games made using HTML, CSS, and JavaScript:

- Rock Paper Scissors  
- Head or Tail (Coin Toss)

A simple home page allows the user to choose between the two games.

## Features

- Clean and responsive design  
- Simple and beginner-friendly code  
- Local storage score saving  
- Auto-play mode for both games  
- Works without any external libraries

## Project Structure

```
index.html                - Home page
rockpaperscissors.html   - Rock Paper Scissors game
rockpaperscissor.css
rockpaperscissor.js
headortail.html          - Head or Tail game
headortail.css
headortail.js
```

## How to Play

### Rock Paper Scissors
Select Rock, Paper, or Scissors. The computer selects its move, and the result is shown instantly.

### Head or Tail
Choose Head or Tail. A virtual coin will flip and show if you won or lost.

## How to Run

1. Download all files.  
2. Open index.html in your browser.  
3. Select a game from the home page and start playing.
